Binning Scheme

We have a number of binning schemes in RNames. The list of binning schemes will be expanded in the future and a feature is in planned to implement binning with users own binning schemes.


Here is the list of currently available schemes:
  • Ordovician Time Slices (Bergström et al. 2009 modified in Rasmussen et al. 2019)
  • Ordovician Time Slices (Webby et al. 2004)
  • Phanerozoic Stages (ICS Chart, 2020)
  • Phanerozoic Epochs (ICS Chart, 2020)

Our default binning algorithm is the combined – algorithm.

Click if you search for binning results from the:

  • shortest – algorithm Each algorithm is linked to page I call “algorithm_window”
  • youngest – algorithm Each algorithm is linked to page I call “algorithm_window”
  • compromise algorithm Each algorithm is linked to page I call “algorithm_window”

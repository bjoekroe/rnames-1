RNames - Help - Wizard

Single page data entry

Enter all data related to one reference on single page

Add a REFERENCE

Add using DOI
  • Insert reference’s DOI to the field and press ’get’
  • Save the reference
Manual entry
  • Press ’manual entry’
  • Fill in the fields and save the reference

Select and Add STRUCTURED NAMES

Select all the structured names mentioned in the reference using the search bar. Create new structured names if they don’t exist already.

Select existing name
  • Search structured names by their name, qualifier or location
Add new structured name
  • Fill in the fields by writing or selecting from the list
  • Save the structured name

Create RELATIONS

Create relations between the structured names that were selected or created in the previous step

Create relation between two structured names
  • Select one structured name from the left field
  • Select another structured name from the right field
Delete relation
  • Similar to creating, select the structured names in the relation from the left and right fields

Once all the data has been added, press 'submit'

If you forgot to add a structured name or a relation but have already submitted the form, check instructions for ’Data entry with three steps’.
